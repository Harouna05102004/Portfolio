Support & Virtualisation

Déploiement GLPI

Contexte de réalisation

Installation d'une solution de gestion de parc informatique (Helpdesk & Inventaire) sur deux environnements distincts pour tester la flexibilité du service.

Missions techniques

  • Cloud Azure : Création d'une machine virtuelle (VM) et configuration des règles de sécurité réseau (NSG) pour autoriser le trafic Web.
  • VirtualBox : Installation locale via une image ISO Debian avec interface graphique pour un environnement de test interne.
  • Stack LAMP : Installation et configuration du serveur Web Apache, de la base de données MariaDB et du moteur PHP.
Déployer un service Virtualisation

Résultat obtenu

Service GLPI accessible via navigateur, prêt pour l'inventaire automatique des postes du domaine efrei.local.
